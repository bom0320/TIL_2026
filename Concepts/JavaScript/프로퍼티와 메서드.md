# 프로퍼티와 메서드

## 1. 프로퍼티란?

프로퍼티는 객체가 가지고 있는 **값이나 정보**이다.

```js
const person = {
  name: "김봄",
  age: 19,
};
```

여기서 `name`, `age` 는 프로퍼티이다.

```js
person.name;
person.age;
```

프로퍼티는 **객체의 상태나 특징**을 나타낸다.

예를 들면:

- 사람의 이름
- 사람의 나이
- 배열의 길이
- 상품의 가격

```js
const list = [1, 2, 3];

console.log(list.length); // 3
```

`length` 는 배열이 가진 정보를 나타내는 프로퍼티

## 2. 메서드란?

메서드는 객체가 가지고 있는 **함수** 이다.
**객체가 할 수 있는 행동**을 나타낸다.

```js
const person = {
  name: "김봄",

  introduce() {
    console.log(`안녕하세요. 저는 ${this.name}입니다.`);
  },
};
```

여기서 `introduce`는 메서드이다.

```js
person.introduce();
```

메서드는 객체를 통해 호출한다.

## 3. 프로퍼티와 메서드의 차이

```js
const person = {
  name: "김봄",

  introduce() {
    console.log("안녕하세요.");
  },
};
```

| 구분     | 예시                 | 의미                 |
| -------- | -------------------- | -------------------- |
| 프로퍼티 | `person.name`        | 객체가 가진 데이터   |
| 메서드   | `person.introduce()` | 객체가 수행하는 행동 |

쉽게 구분하면 다음과 같다.

> 프로퍼티는 객체의 상태이고, 메서드는 객체의 행동이다.

### 둘의 관계

메서드도 넓은 의미에선 프로퍼티이다.

> 프로퍼티 중에서 함수 값을 가진 프로퍼티를 메서드라고 부른다.

즉, 프로퍼티는 객체에 연결된 값이고, 메서드는 그중 함수 값을 가지며 객체를 통해 호출되는 프로퍼티이다.

## 외울 문장

> 프로퍼티는 객체의 상태, 메서드는 객체의 행동이다.

> 메서드는 함수로 된 프로퍼티이다.

> 괄호는 메서드 표시가 아니라 함수를 호출하는 문법이다.
